F1 Dashboard Project

Welcome to the F1 Dashboard! Select a season to view races and their results.

About This Project

This F1 Dashboard is a single-page application (SPA) built with HTML, CSS, and JavaScript. It allows users to explore Formula 1 data for different seasons, including race details, qualifying results, driver information, and constructor standings.

Technologies Used

  • HTML
  • CSS
  • JavaScript
  • Fetch API (for data retrieval)
  • Local Storage (for caching data)
